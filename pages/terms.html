2. Description of the Service

Global Team Clock is a free, browser-based timezone overlap dashboard designed to help remote teams, freelancers, support professionals, and travelers understand and manage time differences across multiple global locations. The Service enables users to add up to ten cities, view their current local times on a visual 24-hour timeline, define custom working hours per location, identify overlapping work windows, and generate shareable configuration links. All functionality executes locally in the user's web browser using the built-in JavaScript Intl API.
